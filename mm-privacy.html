Privacy Policy

Last Updated: October 3, 2025

MemberMessenger respects your privacy. This Privacy Policy explains what data is collected and how it is used:

1. Data Collected

  • Guild (Server) IDs: Used to store configuration (prefix, autoresponses, scheduled messages, etc.).
  • Autoresponse Keywords/Responses: Stored for use within the server where they were created.
  • Scheduled Messages: Stored until they are delivered or removed.
  • Channel IDs: Stored to know where to send scheduled or automated responses.
  • No personal Discord user data (such as email, IP address, or message history) is stored.

2. Data Use

Data is used only to enable features such as autoresponses, scheduled messages, and member/server info. Data is not shared, sold, or used for advertising.

3. Data Storage

Data is stored securely in the Bot’s database. If the Bot is removed from a server, the associated data will not be deleted by default.

4. Data Removal

Server administrators may remove stored data at any time using commands like /removeautoresponse or /removescheduledmessage. Complete data deletion can be requested via the Support Server.

5. Third-Party Services

The Bot interacts with Discord’s API to fetch server and user information. Their use is subject to Discord’s policies:

  • Discord Privacy Policy

6. Changes to this Policy

These terms may be updated from time to time. Continued use of the Bot after updates constitutes acceptance of the new terms.
