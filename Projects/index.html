Anki OVERDRIVE Reverse Engineering

Anki OVERDRIVE is a hybrid racing game where players control miniature cars on modular tracks using mobile devices, integrating physical racing with digital gameplay features like virtual combat and autonomous racing capabilities. Utilizing reverse-engineered protocols, I intend to create software that enables precise control of Anki OVERDRIVE cars from a computer, enhancing functionality beyond the smartphone app's capabilities.
