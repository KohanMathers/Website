Privacy Policy

Last Updated: October 3, 2025

StrikeSystem respects your privacy. This Privacy Policy explains what data is collected and how it is used:

1. Data Collected

  • Guild (Server) IDs: Used to store configuration, allowed roles, and log channels.
  • Moderation Logs: Records of strikes, warnings, mutes, bans, and reasons.
  • User IDs: Stored for moderation tracking (strikes, warnings, mutes, bans).
  • No personal Discord user data (such as email, IP address, or private messages outside of subscribed DMs) is stored.

2. Data Use

Data is used only to enable moderation features and to keep logs of moderation actions. Data is not shared, sold, or used for advertising.

3. Data Storage

Data is stored securely in the Bot’s database. If the Bot is removed from a server, the associated data will not be deleted by default.

4. DM Forwarding

If users subscribe via %subscribedms, their DMs from server members are forwarded to the designated moderators. Users can unsubscribe at any time.

5. Third-Party Services

The Bot interacts with Discord’s API to fetch server and user information. Their use is subject to Discord’s policies:

  • Discord Privacy Policy

6. Changes to this Policy

These terms may be updated from time to time. Continued use of the Bot after updates constitutes acceptance of the new terms.
